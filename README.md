# QA Portfolio

This repository contains sample QA, UAT, release readiness, defect management, Salesforce, and automation-related artifacts designed to demonstrate enterprise-level testing practices and quality governance.

## Repository Structure

### QA Strategy & Governance

Includes artifacts related to QA planning, governance, release readiness, entry and exit criteria, and test closure reporting.

### Test Planning, UAT & Traceability

Includes test plans, UAT plans, business scenarios, test cases, and requirement traceability templates.

### Defect Management & RCA

Includes defect logs, severity and priority matrices, and root cause analysis templates.

### Salesforce QA Assets

Includes Salesforce-focused testing checklists, CPQ validation artifacts, and defect classification templates.

## Purpose

This repository is intended to demonstrate:

* QA strategy and governance
* Test planning and UAT management
* Requirement traceability
* Defect tracking and RCA
* Salesforce QA knowledge
* Release readiness and sign-off
* Business-focused testing practices
* Enterprise quality management

These artifacts are generic, reusable, and suitable for public portfolio use across different industries and delivery models.
